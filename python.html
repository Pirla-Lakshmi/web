  • long

2. Non-Primitive

  • Array
  • Object
  • Class
  • Interface
  • List
  • Set
  • Dictionary

Operators

There are 5 operators:

  • Arithmetic → + - * / %
  • Relational → == != > < >= <=
  • Logical → and or not
  • Assignment → = += -= *= /= %=
  • Bitwise → & | ^ ~ << >>

Conditional Statements

There are 5 conditional statements:

  • elif
  • if-else
  • Nested if
  • Ternary (if-else)
